About Me

I am a former grad student that enjoys creating and solving mathematical puzzles. Differential equations are my jam! My research at McMaster was in the global dynamics of a predator-prey model in a chemostat. Specifically, I performed asymptotic and bifurcation analysis on this model with a Holling Type II predatory response function. In the past I have also assisted in the research of integrable Non-Abelian Laurent ODEs.

Floodfill

Floodfill is a web game I helped create for Caribou Contests as an application of the Four Colour Theorem.

Status: Completed

My Roles:

  • Designed the interface to load a puzzle from the database and draw its regions onto an HTML5 canvas
  • Implemented the Fabric.js library to allow user interaction with the canvas
  • Developed scoring procedures to check if a puzzle has been solved

Calcrostic Puzzles

Calcrostics are puzzles I helped create for Caribou Contests that show calculations in a form similar to acrostic word puzzles.

Status: Completed

My Roles:

  • Designed the interface to load a puzzle from the database and format its equations inside user-interactable textboxes
  • Implemented an HTML5 canvas to display when an equation is correct
  • Developed scoring procedures to check if a puzzle has been fully solved

Knot Theory Mini Course

I am currently developing a mini course for Caribou Contests to teach students the basics of knot theory, along with an interactive workbench for them to draw and "untangle" their own knots.

Status: In Progress

My Roles:

  • Researching knot theory to formulate a lesson plan for high school students
  • Designing an interface to draw and manipulate knots on an HTML5 canvas
  • Developing procedures to automate Reidemeister Moves
